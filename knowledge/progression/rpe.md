# RPE (Rating of Perceived Exertion)

## Definition

RPE is a subjective measure of how hard a set feels on a scale of 1-10.

For resistance training, RPE is typically defined relative to **reps in reserve (RIR)**:

**RPE = 10 - RIR**

## RPE Scale

| RPE | Description | Reps Left | Effort |
|-----|-------------|-----------|--------|
| 10 | Maximum — cannot do another rep | 0 RIR | Absolute failure |
| 9.5 | Could not perform another rep, maybe could grind 0.5 | ~0-1 RIR | Near failure |
| 9 | Very challenging — 1 rep left | 1 RIR | Heavy, safe |
| 8.5 | 1-2 reps left | ~1-2 RIR | Heavy with control |
| 8 | 2 reps left | 2 RIR | Good intensity |
| 7.5 | 2-3 reps left | ~2-3 RIR | Moderate-hard |
| 7 | 3 reps left | 3 RIR | Moderate |
| 6 | 4+ reps left | 4+ RIR | Light working |
| 5 | Warm-up | ∞ | Not taxing |

## RPE Targets for Hypertrophy

| Exercise Type | RPE Target | Equivalent RIR |
|--------------|------------|----------------|
| Primary compound | 7-8 | 2-3 RIR |
| Secondary compound | 8 | 1-2 RIR |
| Machine compound | 8-8.5 | 1.5-2 RIR |
| Isolation | 8-9 | 1-2 RIR |
| Final set (isolation) | 9-10 | 0-1 RIR |

## Why RPE Instead of Percentages?

| Method | Pros | Cons |
|--------|------|------|
| RPE (auto-regulation) | Adapts to daily readiness | Subjective, requires experience |
| Percentage of 1RM | Objective, measurable | Doesn't account for fatigue |

GymOS uses RPE as the PRIMARY intensity tool because:
- The user primarily trains for hypertrophy, not max strength
- Auto-regulation prevents overtraining on bad days
- RPE adapts to sleep, stress, and nutrition fluctuations

## RPE Session Examples

### Compound Exercise (RPE 8)
- Warm-up: bar × 10, 40% × 5, 60% × 3
- Set 1: Working weight × target reps (RPE 7-8)
- Set 2: Same weight × target reps (RPE 8)
- Set 3: Same weight × target reps (RPE 8-8.5)
- Set 4: Same weight × target reps (RPE 8.5-9)

### Isolation Exercise (RPE 8-9)
- Set 1: Working weight × target reps (RPE 8)
- Set 2: Working weight × target reps (RPE 8-9)
- Set 3: Working weight × target reps (RPE 9)
- Set 4: Working weight × target reps (RPE 9-10) — optional final set to failure
