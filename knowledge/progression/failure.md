# Training to Failure

## Definition

**Technical failure:** The point at which you cannot complete another rep with proper form despite maximal effort.

**Absolute failure:** The point at which the muscle cannot produce enough force to move the weight through any part of the range of motion.

**GymOS uses technical failure as the definition.** When form breaks down significantly, the set is over.

## Guidelines

| Exercise Type | Train to Failure? | Frequency |
|--------------|-------------------|-----------|
| Compound free weight (bench, squat, OHP, row) | Not recommended | Rarely, if ever |
| Machine compound (chest press, lat pulldown) | Occasionally | 1-2 final sets per session |
| Isolation (curls, raises, extensions) | Yes, final sets | Last working set |
| Abs / calves | Yes | Final sets |
| Warm-up | Never | — |

## When to Push to Failure

- Final set of an isolation exercise (last set of lateral raises, curls, etc.)
- Last session of a mesocycle before deload
- When testing a new rep max (rare)
- When specifically targeting metabolic stress/hypertrophy stimulus

## When to AVOID Failure

- Every set of every exercise (excessive CNS fatigue)
- Heavy compounds (injury risk, systemic fatigue)
- Early in the session (will impair subsequent exercises)
- During a deload or recovery week
- When technique is still being learned

## The Cost of Failure

| Consequence | Impact |
|-------------|--------|
| CNS fatigue | High — can impair performance for days |
| Recovery time | Longer — failure sets take longer to recover from |
| Injury risk | Higher — form breakdown under heavy loads |
| Session impact | Less total volume (fewer sets possible after failure set) |

## Recommended Approach (Hypertrophy)

- **Most sets:** 1-3 reps from failure (RPE 7-9)
- **Last set of isolation:** Can go to 0-1 RIR (near failure)
- **Once per exercise per session:** Only the final set should approach failure
- **Never on compounds with a spotter** unless spotter is present
- **When in doubt, leave one rep in the tank**
