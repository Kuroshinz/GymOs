# Double Progression — NEXUS Default Method

## Concept

Two variables to progress: **reps first, then weight**.

## How It Works

1. Pick an exercise with a target rep range (e.g., 8–12)
2. Start at a weight you can hit 8 reps with
3. Each session, try to add reps while staying in the range
4. Once you hit the top of the range (12 reps) for ALL sets
5. Increase weight by the smallest increment
6. You'll likely drop back to ~8 reps — repeat the cycle

## Example

| Session | Sets x Reps | Weight |
|---------|------------|--------|
| 1 | 3 x 8, 8, 8 | 50kg |
| 2 | 3 x 9, 8, 8 | 50kg |
| 3 | 3 x 9, 9, 9 | 50kg |
| 4 | 3 x 10, 10, 9 | 50kg |
| 5 | 3 x 10, 10, 10 | 50kg |
| 6 | 3 x 11, 11, 10 | 50kg |
| 7 | 3 x 12, 11, 11 | 50kg |
| 8 | 3 x 12, 12, 12 | 50kg |
| 9 | 3 x 8, 8, 8 | 52.5kg 🎉 |

## Why This Works

- **Progressive overload without guessing**: clear rule for when to add weight
- **Volume accumulation**: more reps = more volume = more stimulus
- **Skill practice**: more time at submaximal weights to hone technique
- **Injury prevention**: never adding weight until you've mastered the current load

## Rep Ranges by Goal

| Goal | Compound | Accessory |
|------|----------|-----------|
| Strength | 3–6 | 5–8 |
| Hypertrophy | 6–10 | 8–15 |
| Endurance | 10–15 | 15–20 |

## When to Progress Weight

Progress only when you hit the top of the rep range on **all sets** in a session — not just the first set. This ensures sufficient volume accumulation.
