# User Training Configuration

## Split: PPL-UL

Push / Pull / Legs / Upper / Lower rotation.

### Schedule

| Day | Session | Priority |
|-----|---------|----------|
| 1 | Push A | Upper chest / shoulders focus |
| 2 | Pull A | Width focus (lats, rear delts) |
| 3 | Legs A | Quad-dominant |
| 4 | Upper A | Overhead emphasis, heavier |
| 5 | Lower A | Posterior chain |
| 6 | Push B | Volume focus, isolations |
| 7 | Pull B | Horizontal + vertical pull |
| 8 | Legs B | Hamstring-dominant |
| 9 | Upper B | Volume, pump work |
| 10 | Lower B | Quad + glute volume |

Rest days taken as needed (typically after 2-3 sessions).

### Training Parameters

| Parameter | Value |
|-----------|-------|
| Rep range (compounds) | 8-12 |
| Rep range (isolation) | 10-15 |
| Rest between sets | 60-90 seconds |
| RPE target (compounds) | 7-8 (2-3 RIR) |
| RPE target (isolation) | 8-9 (1-2 RIR, last set to failure) |
| Eccentric tempo | 2-3 seconds |
| Concentric tempo | 1-2 seconds |
| Warm-up | 5-10 min cardio + dynamic stretching + specific warm-up |

### Progression Method

**Double Progression:**
1. Stay at same weight until hitting top of rep range on all sets
2. Increase weight by smallest increment (1.25-2.5 kg upper, 2.5-5 kg lower)
3. Repeat

### Deload

Every 6-8 weeks or when:
- Progress stalls for 2+ consecutive sessions
- RPE feels unusually high
- Sleep/mood/motivation drops
- Returning from layoff

Deload: 40-60% volume reduction, same intensity, same frequency.

## Focus Muscles

| Priority | Muscle | Why |
|----------|--------|-----|
| 1 | Side Delts | Lateral raises — most visible for width |
| 2 | Rear Delts | Face pulls, reverse flies — shoulder health + fullness |
| 3 | Upper Chest | Incline press, low-to-high fly — "armour plate" |
| 4 | Lats (width) | Pulldowns, pull-ups, wide rows — V-taper |
| 5 | Arms | Biceps + triceps — proportional aesthetics |
